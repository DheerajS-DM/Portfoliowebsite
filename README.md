🚀 Tech Stack
Frontend: React 19

Styling: Tailwind CSS (v4)

Bundler: Vite

Deployment: Vercel

🕹️ Key Features
Custom Game Engine: A native Tetris implementation built from scratch using React hooks (no external game libraries).

Responsive 70/30 Layout: Dynamic sidebar integration that toggles between a professional project showcase and an interactive arcade.

Backend-First Design: Optimized project cards designed to highlight core logic, research, and CLI tools even without live deployment links.

Glassmorphism UI: Modern, dark-mode aesthetic with animated mesh gradients and backdrop blurs.

Installation & Setup
Clone the repository:

Bash
```
git clone https://github.com/DheerajS-DM/portfolio.git
Install dependencies:
```
Bash
```
npm install
```
Run locally:
Bash
```
npm run dev
```
Build for production:
Bash
```
npm run build
```
Project Structure
src/App.jsx - Main layout and project data.

src/tetris.jsx - The core game logic and rendering engine.

src/index.css - Tailwind configurations and global animations.
